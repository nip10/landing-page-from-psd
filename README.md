## Getting Started

### Pre-requisites

* Node

### Installing

* Install packages

```
npm install
```

* Build assets

```
npm run build
```

## Deployment

Build assets

```
npm run build:prod
```

## Built With

* HTML5
* SASS (using BEM methodology)
* ES6
